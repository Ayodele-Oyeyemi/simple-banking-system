Simple Banking System (C#)

This is a simple console-based banking system built with C# and .NET.
The project simulates basic banking operations and was created to practice object-oriented programming and core C# concepts.

The application runs entirely in the terminal and allows users to create and manage bank accounts through a menu-driven interface.

Features

Create a new bank account

Deposit money into an account

Withdraw money from an account

Check account balance

Basic error handling for invalid inputs

Project Structure
SimpleBankingSystem/
│── Program.cs        // Entry point and menu logic
│── Account.cs        // Account model and operations
│── Bank.cs           // Manages multiple accounts
│── SimpleBankingSystem.csproj

Technologies Used

C#

.NET (Console Application)

Visual Studio Code

How to Run the Project

Make sure you have .NET SDK installed

Clone the repository

Navigate into the project folder:

cd SimpleBankingSystem


Run the application:

dotnet run

Sample Usage

When the program starts, you’ll see a menu where you can:

Create an account

Enter an account number to deposit or withdraw

Check balances in real time

The application runs until you choose to exit.
